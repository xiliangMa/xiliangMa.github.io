# iptables 基础操作

#### 1. 查看规则列表

```
iptables -nvL
```

#### 2. 清空防火墙规则

```
iptables -F
```

#### 3. 允许22端口

```
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
```
